Multiply by itself

In this tutorial, we will be using classes, arrow functions, instances, the return statement, and the this object to get the area of a square.

The advanced section expect you to have finished previous sections. Concepts described there won't be revised here.

Concepts Used

Classes

Classes are special functions that serve as blueprints to create an instance, which is the output of the class. For example, the class could be meant to create a dog, and the instance would be a dog based on the class with its own methods and parameters.

class Dog { constructor(personality) { this.personality = personality } getPersonality = () =>{ return this.personality } } console.log(new Dog("happy").getPersonality())

This code creates a class called Dog, which has a constructor called personality, which assigns the this object's personality portion to the dog's personality for future usage globally across the class. Constructors are parts of your blueprint (class here). For example, our dog's personality should be a constructor, because it's a defining part of it. Then we create a function called getPersonality, which refers to the this.personality we assigned earlier and returns it. Then, we simply print its personality into the console after creating a new dog, assigning it a personality, and retrieving it.

This object

The this object is global, and refers to the object or instance that executed the code, shown in the earlier classes example.

class Dog { constructor(personality) { this.personality = personality

// The this object is assigned here!

} getPersonality = () =>{ return this.personality } } console.log(new Dog("happy").getPersonality())

Get the area

Now to mesh these concepts together.

The following code assigns a class called Square. One of its constructors, side is defined, and the side property of the this object is set to the side defined by the user (what the constructor's side field is) for later usage across different functions inside the class. Then we create an arrow function called getSide(), which simply returns the this.side property. Then, we define a new variable called getArea(), which is equal to a regular unnamed function. This function then returns yet another function, which returns the this.side property we defined globally earlier multiplied by itself to get the area. Then we print the value of the side of the square, and the square's area into the console. That's it!

class Square { constructor(side){ this.side = side } getSide = () =>{ return this.side } getArea = function(){ return () => { return this.side * this.side } } } console.log(`side of square: ${new Square(21).getSide()}`) console.log(`area of square: ${new Square(5).getArea()()}`)
