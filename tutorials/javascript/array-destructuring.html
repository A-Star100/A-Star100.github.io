Any Animal

In this tutorial, we will be using array destructuring and variables to display a portion of our animals to the user.

Basic Concepts Used

How to use variables (constants)

Variables can keep track of values for you, but constants cannot be changed.

const myName = "John Doe" console.log(`Hello, ${myName}!`)

How to use arrays

Arrays are lists that can contain anything.

let numbers = [1, 2, 3, 4, 5] console.log(`Numbers: ${numbers}`)

How to use array destructuring

Arrays destructuring is a way to get values from an array with only a bit of syntax

const fruits = ["apple", "peach", "orange", "grapes", "blueberry"]

// The ",," means that we skip an item in the list, and the "..." means that we get the rest of the list as an array, and we assign the fruits retrieved to three different variables and print them.

const [a,,b,...c] = fruits console.log(a, b, c)

Any Animal

Now, to mesh these concepts together. Let's call out our animals!

const animals = ["snake", "tiger", "lion", "giraffe", "hippo"] const [a,,b,...c] = animals console.log(a, b, c)
