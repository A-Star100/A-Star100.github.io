Favorite Fruits

In this tutorial, we will be using basic array methods along with functions and variables to make a list of your favorite fruits.

Basic Concepts Used

How to use variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello, ${myName}!`)

How to use functions

Functions can execute things when you tell them to and accepts inputs.

function greetPerson(person) { console.log(`Hello, ${person}!`) } greetPerson("John Doe")

How to use arrays

Arrays are lists that can contain anything.

let numbers = [1, 2, 3, 4, 5] console.log(`Numbers: ${numbers}`)

Favorite Fruits

Now, to mesh these concepts together. Let's create our lists of fruits so you can enjoy them after you understand this code.

let fruits = ["apple", "banana", "grape"] fruits.push("mango")

// The push operation adds a new item to our list

console.log(fruits)

Hint: Try changing the fruit names in the array, or add more with .push("any-fruit")!
