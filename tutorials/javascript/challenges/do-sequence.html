Functions can execute things when you tell them to and accepts inputs.

Greet a person

Now to mesh these concepts together.

We will be using variables to greet a person named John Doe with a friendly hello.

let name = "John Doe" console.log(`Hello there, ${name}!`)

This code simply greets John Doe with a friendly "Hello there!" in the console.

**Think:** Instead of saying hello at a fixed time, why not be able to say hello anytime. Once you've figured out an answer, click "See answer".

function greetPerson(name) { console.log(`Hello there, ${name}!`) } greetPerson("Jane Doe")

This code instead uses a function that accepts a name as an input. When the function is called with "greetPerson()", it greets Jane Doe in the developer's console.
