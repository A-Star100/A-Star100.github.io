Now to mesh these concepts together.

We will be using for loops and an arrow function to perform a sequence with number patterns. We first define our function as a variable (changeable), with three values, the number we start at, the value we increment by, and the number we end at. Then we create our loop. We set our index value (i) to the start value provided by the user. The loop will keep running as long as our index (position) is less than or equal to the end value, and our index value will increment the same amount as the inc value to keep the index value correct regardless of how much we increment. That's it!

let doSequence = (start, inc, end) => { for (let i = start; i <= end; i = i + inc ){ console.log(i) } } doSequence(3, 6, 9)
