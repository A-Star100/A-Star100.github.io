Find the largest number

Now to mesh these concepts together.

First, we will define a list of numbers. Then, we will define a variable called maxNum, our largest number. Think of a few pairs of socks as our numbers. In the for loop we define next, we will compare one sock against the other. Whichever sock is bigger is regarded as the largest number (maxNum). We will keep doing this until the end of the list of numbers and then report the largest number to the user.

const numbers = [1, 2, 3, 4, 5] let maxNum = 0 for (let i = 0; i < numbers.length; i++) { if (maxNum = 0) { if (numbers[i] < numbers[i + 1]) { maxNum = numbers[i + 1] } } else { if (maxNum < numbers[i]) { maxNum = numbers[i] } } } console.log(`largest number: ${maxNum}`)

This code has a list of numbers and our largest number. It checks every number until the position of which number we're checking is greater than the length of the list itself (meaning the list is over). If our largest number variable is 0, we compare the current number we're checking with the next number. If the next number is larger, we set the largest number so far to the next number. Else we go on. At the end, we tell the user the largest number.

This code instead defines an array first and keeps the largest number variable inside the function where we check the largest number to keep from confusion. Instead of checking if the largest number is 0, we instead check if the current number we're checking is greater than the largest currently defined at this point. Then we set the largest number to the current number we're checking. Then, we return the largest number at the end to print into the console.
