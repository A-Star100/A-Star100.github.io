Find the average

In this tutorial, we will be using the reduce method, variables, arrays, and arrow functions to find the average score out of 11 scores from students on a test.

Concepts Used

How to use variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello, ${myName}!`)

How to use arrow functions

Arrow functions can execute things when you tell them to and accept inputs, but they are formatted differently.

let greetPerson = (person) => console.log(`Hello, ${person}!`); greetPerson("John Doe")

How to use the reduce method

The reduce method is similar to a for loop but it has built in tools equipped to reduce several numbers into one value, making it preferred for tasks like averaging numbers in JavaScript.

const numbers = [15.5, 2.3, 1.1, 4.7]; console.log(numbers.reduce(getSum, 0)) function getSum(total, num) { return total + Math.round(num); }

How to use arrays

Arrays are lists that can contain anything.

let numbers = [1, 2, 3, 4, 5] console.log(`Numbers: ${numbers}`)

Find the average

Now to mesh these concepts together.

First, we define our scores from the students, and define an iteration variable (keeps track of our position (a.k.a index) since we aren't using a for loop). Then we add two variables, total and current. The reduce block acts as a for loop and gives us our first value. The current value is treated as the second number we encounter while the total starts from the first number and the return block adds these two values.

let scores = [10, 75, 83, 63, 21, 99, 11, 51, 45, 63, 88] let iteration = 0 let sum = scores.reduce((total, current) => { console.log(`iteration: ${iteration}, total: ${total}, current: ${current}`) iteration++ return total + current }) let average = sum / scores.length console.log(`Average score: ${average}`)
