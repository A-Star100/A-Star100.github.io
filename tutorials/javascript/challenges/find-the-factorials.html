Find the factorials

In this tutorial, we will be using a for loop, variables, and functions to figure out the factorial of a number.

How to use functions

Functions can execute things when you tell them to and accepts inputs.

How to use variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello there, ${myName}!`)

Find the factorials

Now to mesh these concepts together.

This code first defines a function called factorial(), then it defines a variable called result. Then it uses a for loop and tracks the index of where we are. The index is set to one. As long as the index is less than or equal to the number we specified, the loop keeps running. The result is multiplied by the index. The index will keep increasing with every iteration (for example, 1 * 1, then 1 * 2).

function factorial(num) { let result = 1; for (let i = 1; i <= num; i++) { result = result * i } return result } console.log(`factorial of 5: ${factorial(5)}`) console.log(`factorial of 2: ${factorial(2)}`) console.log(`factorial of 6: ${factorial(6)}`) console.log(`factorial of 8: ${factorial(8)}`)
