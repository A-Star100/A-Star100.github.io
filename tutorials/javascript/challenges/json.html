Show the scores

Now to mesh these concepts together.

We will be using variables and JSON to display all the students' scores neatly printed into the console. The code first defines a constant containing our JSON data, then creates students and assigns them scores and courses they're registered to. Afterwards, we simply print all the data as an object and print everybody's score. That's it!

/* There are 4 students each enrolled in 2 courses, Computer Science and English. These students are assigned with random scores. */

const scores = { John: { score: 91, courses: ["Computer Science", "English"] }, Bob: { score: 92, courses: ["Computer Science", "English"] }, Arnold: { score: 94, courses: ["Computer Science", "English"] }, Joe: { score: 89, courses: ["Computer Science", "English"] } } console.log(scores) console.log(`John's score: ${scores.John.score}`) console.log(`Bob's score: ${scores.Bob.score}`) console.log(`Arnold's score: ${scores.Arnold.score}`) console.log(`Joe's score: ${scores.Joe.score}`)
