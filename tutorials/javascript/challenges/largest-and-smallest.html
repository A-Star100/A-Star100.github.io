Find the largest even number and smallest odd number in an array

In this tutorial, we will be using for loops and arrays to determine the largest even number and smallest odd number in a list.

Find the numbers

Now to mesh these concepts together.

First, we will define two variables, largestEven and smallestOdd, along with our array of numbers. For each number in the list, we first check if it is even or odd by dividing it by 2 using modulus (modulus is a math operation that gets the remainder of a division problem, if the remainder is 0 it is even).

let numbers = [23,4,71,16,18,52,4,8,21,96,126,34,22,5] let largestEven = null; let smallestOdd = null; for (let i = 0; i < numbers.length; i++) { if (numbers[i] % 2 == 0) { if (numbers[i] > largestEven) { largestEven = numbers[i] } } else { if (smallestOdd === null) { smallestOdd = numbers[i] } if (numbers[i] < smallestOdd) { smallestOdd = numbers[i] } } } console.log(`Largest even: ${largestEven}, smallest odd: ${smallestOdd}`)
