Calculate with stocks

In this tutorial, we will be using objects within arrays, variables, arrow functions, loops, and more to calculate with our stocks.

Concepts Used

Arrays

Learn more about arrays here.

Objects

Objects can be used to access values in variables, and in our case, serve as formatted data.

const array1 = [{ value1: "123", value2: 400, value3: 75, value4: 376 }, { value1: "1233", value2: 4030, value3: 735, value4: 3736 }

// The above two wrapped in curly brackets ({/}) are series' of objects inside an array. ]

Variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello, ${myName}!`)

Loops

Learn more about loops here.

forEach loop

A forEach loop iterates through each item in a list and assigns the current values to a variable of your choice. This is useful for list-based loop interaction with less lines of code.

let words = ['Amazing', 'Cool', 'Great'] words.forEach((word) => console.log(word))

Object Access Notation

We can specify the variable containing the objects and its property separated by a dot to access the value. We can access the same value in other ways too, but for simplicity we'll choose this notation.

console.log(snake.venom)

Calculate with stocks

Now to mesh these concepts together.

We will be using variables, objects, arrays, and loops to calculate the what we paid for our stocks, how much we can sell them for now, and the profit we'll make if we do sell them. That's it!

const stocks = [ { ticker: "123", quantity: 400, cost: 75, price: 376 }, { ticker: "456", quantity: 2000, cost: 36, price: 25 }, { ticker: "789", quantity: 50, cost: 100, price: 20 }, { ticker: "987", quantity: 1500, cost: 160, price: 10 } ] let totalCost = 0; let totalPrice = 0; for (let i = 0; i < stocks.length; i++) { const stock = stocks[i] totalCost += stock.cost * stock.quantity totalPrice += stock.price * stock.quantity } console.log(`total cost: ${totalCost}, total price: ${totalPrice}, total profit: ${Math.abs(totalPrice - totalCost)}`)
