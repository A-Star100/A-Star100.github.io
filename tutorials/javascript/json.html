Keep safe from the snake

In this tutorial, we will be using JSON objects, object access, variables and if statements to check if the snake we defined is venomous or not, and if it is, we will notify the user.

Basic Concepts Used

Variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello, ${myName}!`)

If statements

Learn more about the if statement here

Basic JSON Object Access Notation

We can specify the variable containing JSON and its property separated by a dot to access the value. We can access the same value in other ways too, but for simplicity we'll choose this notation.

console.log(snake.venom)

Keep safe from the snake

Now to mesh these concepts together.

We first create our snake and assign properties to it. Then we use a basic if statement to check if the venom property of the snake is true or false. Depending on the value, we notify the user with different messages.

let snake = { speak: "hiss", color: "green", venom: true } if (snake.venom === false) { console.log("You are sssss-safe") } else { console.log("You are sssss-going to dieeeeeeeeee") } console.log(snake)
