Loops

In this tutorial, we will learn how to use for loops, while loops, and break/return statements with them in JavaScript

Concepts Used

For loops

For loops iterate through a list of items. This can help with checking values in lists and save time.

const numbersList = [1000, 81, 23, 5, 6, 101] function getLargestNumber(numbers) { let max = 0 for (let i = 0; i < numbers.length; i++){ if (numbers[i] > max) { max = numbers[i] } } return max } console.log(`Largest number: ${getLargestNumber(numbersList)}`)

This code uses a for loop to iterate through a list of numbers, and check which is the largest by comparing them to the largest one encountered so far.

How to use arrays

Arrays are lists that can contain anything.

let numbers = [1, 2, 3, 4, 5] console.log(`Numbers: ${numbers}`)

How to use break and return

"break" and "return" are two JavaScript blocks that end a loop and return a value respectively. "break" is the only one that can break out of any loop, even a "while true" one.
