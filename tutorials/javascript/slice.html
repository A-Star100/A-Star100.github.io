Split words

In this tutorial, we will split a word even in length in half and show the user the first half.

Concepts Used

How to use variables

Variables can keep track of values for you.

let myName = "John Doe" console.log(`Hello, ${myName}!`)

How to use functions

Functions can execute things when you tell them to and accepts inputs.

function joinStrings(string, string2) { console.log("Answer:" + string + string2) } joinStrings('Hello', 'world')

Split the word in half

Now to mesh these concepts together.

We will be using a function and modulus (modulus is a math operation that returns the remainder of a division problem) to detect if a string is even, and if so, split it in half and give it back to the user.

function firstHalf(word) { if (word.length % 2 == 0) { const halfWord = word.length / 2; const result = word.slice(0, halfWord); return result; } else { return "This string is not even, so it cannot be split into half" } } console.log(firstHalf("WooHoo"))

This code splits the even string "WooHoo" into half and tells the user the first half.

Tip: You can remove the "0" (position in word length to start splitting the half from) to get the last half instead of the first half as the halfWord variable is the position to start spliting from (with a 0 it becomes the position to end splitting).
