MOBJs

In this tutorial, we will be using basic SRB2 Lua MOBJs to spawn a balloon in the air..

Basic Concepts Used

Hooks

Hooks are like event listeners, but trigger in the middle or between tics instead of every overall tic.

addHook("ThinkFrame", function() print("Hello world")

-- This prints "Hello world" into the console pretty much every frame

end)

Freeslots

Freeslots are empty slots in SRB2 for things such as enemies and more. You can create your own slot and assign it a value.

freeslot("MT_DUMMY") mobjinfo[MT_DUMMY] = {

-- create an extremely basic object

spawnhealth = 1, spawnstate = S_INVISIBLE } mobjinfo[MT_DUMMY].hello = "hi!"

-- custom mobjinfo field

MOBJs

Now to mesh these concepts together.

In this code, it assigns a new, blank MOBJ to a freeslot. The MOBJ is invisible but still spawned in-game. If you use existing MOBJs, like MT_CRAWLA, you can spawn enemies and more in-game. The code is mostly self-explanatory.

-- credit to Make new variables when defining a mobj thread on SRB2 MB

freeslot("MT_DUMMY") mobjinfo[MT_DUMMY] = { spawnhealth = 1, spawnstate = S_INVISIBLE } mobjinfo[MT_DUMMY].hello = "hi!" addHook("MobjThinker", function(mobj) print( mobj.info.hello ) end, MT_DUMMY)
