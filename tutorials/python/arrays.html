Basic Concepts Used

How to use variables

Variables help keep track of values.

my_name = "John Doe" print(f"Hello, {my_name}!")

How to use functions

Functions execute actions and can accept inputs.

def greet_person(person): print(f"Hello, {person}!") greet_person("John Doe")

How to use for loops

Loops repeat actions a number of times. Learn more about loops here

How to use arrays (lists)

Lists store collections of values.

numbers = [1, 2, 3, 4, 5] print("Numbers:", numbers)

Favorite Fruits

Now let's mesh these concepts together to build a favorite fruit list!

fruits = ["apple", "banana", "grape"] fruits.append("mango") # The append() method adds an item to the list print(fruits)
