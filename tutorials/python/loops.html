Loops

In this tutorial, we will learn how to use for loops, while loops, and break/return in Python.

Concepts Used

For loops

For loops iterate through a list of items. This can help with checking values in lists and save time.

numbers_list = [1000, 81, 23, 5, 6, 101] def get_largest_number(numbers): max_num = 0 for num in numbers: if num > max_num: max_num = num return max_num print(f"Largest number: {get_largest_number(numbers_list)}")

This code uses a for loop to iterate through a list of numbers, and find the largest by comparing each one to the biggest found so far.

While loops

While loops run as long as a condition is true.

age = 10 counter = 0 while age == 10 and counter < 3: print("I am 10 years old") counter += 1 # Simulated delay import time time.sleep(3) age = 11 print("I am 11 years old")

Lists (Arrays)

Lists can contain anything: numbers, strings, even other lists.

numbers = [1, 2, 3, 4, 5] print(f"Numbers: {numbers}")

break and return

break ends a loop early. return exits a function and can return a value.

age = 10 while age == 10: print("I am 10 years old") break numbers_list = [1000, 81, 23, 5, 6, 101] def get_largest_number(numbers): max_num = 0 for num in numbers: if num > max_num: max_num = num return max_num
